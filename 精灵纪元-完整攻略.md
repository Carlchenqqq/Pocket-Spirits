## 📖 游戏简介

《精灵纪元》是一款东方幻想风格的精灵收集对战RPG游戏。在这个名为**九州灵域**的世界中，你将扮演一名16岁的灵师学徒，与各种灵兽缔结契约，探索广袤的世界，挑战道馆，揭开浊流组织的阴谋。

### 各区域详解

#### 🏠 青叶镇（出生地）
- **功能**：新手村、领取初始精灵
- **NPC**：
  - 墨博士（研究所）- 发放初始精灵
  - 芸娘（家中）- 母亲
  - 张大妈、小明的小伙伴 - 路人对话
- **出口**：南 → 青叶野外

#### 🌿 青叶野外（新手区域）
- **野生精灵**：
  - 雷鹰 Lv.2-4 (30%)
  - 幻影龙 Lv.2-3 (25%)
  - 超能猫 Lv.3-4 (15%)
  - 念力熊 Lv.3-4 (10%)
- **训练师**：
  - 少年训练师阿明 - 雷鹰Lv.3 + 幻影龙Lv.4
  - 少女训练师小芸 - 雷鹰Lv.4 + 超能猫Lv.5
- **出口**：西 → 青叶镇 | 东 → 碧波森林

#### 🌲 碧波森林
- **特点**：森林迷宫，道路曲折
- **野生精灵**：
  - 芽叶兔 Lv.4-6
  - 雷鹰 Lv.4-6
  - 暗影蝠 Lv.4-6
  - 小石灵 Lv.3-5
  - 花仙子 Lv.5-7 (稀有)
- **出口**：西 → 青叶野外 | 东 → 碧波镇

#### 🏘️ 碧波镇（第一个城镇）
- **功能**：商店、治疗、道馆
- **NPC**：
  - 货郎老黄 - 商店
  - 杏林医仙 - 免费治疗
  - 港口船夫 - 剧情提示
- **连接**：
  - 西 → 碧波森林
  - 北 → 迷雾沼泽
  - 南 → 礁石航道
  - 东 → 赤岩古道

#### 🌫️ 迷雾沼泽
- **特点**：沼泽地形，视野受限
- **野生精灵**：
  - 泥潭怪 Lv.10-14
  - 暗影蝠 Lv.10-14
  - 幽冥狼 Lv.12-16
  - 水母灵 Lv.10-13
  - 九命灵猫 Lv.14-16 (极稀有)
- **特殊**：灵晶碎片收集地

#### 🏔️ 赤岩古道
- **特点**：山地地形，火系精灵聚集地
- **野生精灵**：
  - 小石灵 Lv.12-16
  - 岩巨人 Lv.13-17
  - 火花犬 Lv.12-15
  - 沙蠕虫 Lv.14-16
  - 熔岩蟹 Lv.15-18

#### 🌊 礁石航道
- **特点**：海上路线，需剧情解锁
- **野生精灵**：
  - 水母灵 Lv.8-12
  - 贝壳精 Lv.8-11
  - 萤火虫 Lv.6-10
  - 深渊鳗 Lv.12-15 (极稀有)

### 实用克制口诀
- **火克草、冰** → 打火系道馆别用草/冰
- **水克火、岩** → 水系是火岩的克星
- **草克水、岩** → 草系对抗水系优势
- **电克水** → 电系打水系双倍伤害
- **岩克火、冰** → 岩石系克制火冰
- **冰克草、龙** → 冰系是龙系克星
- **暗克超能** → 暗影对超能效果绝佳
- **超能克暗无效** → 超能打暗系无效！

## 🎯 战斗技巧

### 1. 属性克制优先
战斗前先观察对方属性，选择克制属性的技能可造成双倍伤害！

### 2. 速度决定先手
速度高的精灵先出手，可以抢先击败对手或施加状态。

### 3. 状态技能运用
- **叫声**：降低对方攻击，适合持久战
- **变硬**：提升防御，适合坦克型精灵
- **龙之舞**：攻防速全提升，龙系神技

### 4. 捕捉技巧
- 先削弱对方HP（红色血条时捕捉率最高）
- 使用状态技能（麻痹/睡眠可提升捕捉率）
- 根据稀有度选择精灵球

### 5. 队伍搭配建议
推荐配置：
- 1只高速攻击手（如焰尾狐、雷电鼠）
- 1只防御坦克（如岩甲兽、铁壁龟）
- 1只特攻手（如超能猫、花仙子）
- 属性尽量覆盖全面

## 📜 主线流程攻略

### 第一章：灵师之路启程

**目标**：获得初始精灵 → 前往碧波镇

1. **青叶镇**
   - 与墨博士对话，选择御三家
   - 与母亲告别
   - 向南出发

2. **青叶野外**
   - 练习战斗，捕捉1-2只精灵
   - 击败少年训练师阿明
   - 击败少女训练师小芸
   - 向东进入碧波森林

3. **碧波森林**
   - 穿越森林迷宫
   - 捕捉草系精灵补充队伍
   - 向东到达碧波镇

4. **碧波镇**
   - 与货郎老黄对话购买道具
   - 找杏林医仙治疗
   - 探索城镇，与NPC对话
   - 准备挑战道馆（V2开放）
